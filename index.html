Строим станции: структура HTML

Как станции метро соединены тоннелями, так HTML-элементы образуют структуру веб-страницы.

Каркас станции (базовая структура HTML):


<!DOCTYPE html>
<html lang="ru">
<head>
    <meta charset="UTF-8">
    <title>Название станции</title>
</head>
<body>
    <header>Вход на станцию</header>
    <main>Центральный зал</main>
    <footer>Выход к тоннелям</footer>
</body>
</html>

Основные элементы - жители метро

<h1>-<h6>

Как указатели станций - определяют важность заголовков

<p>

Как рассказы старых сталкеров - содержат текстовую информацию

<a>

Как переходы между станциями - связывают страницы

<img>

Как фотографии на стенах - показывают изображения

Оформление станций: сила CSS

Как фонарь Артёма освещает темные тоннели, CSS освещает и украшает ваши HTML-страницы.

Пример освещения (базовые стили):


/* Как газовые лампы станции */
body {
    background-color: #1a1a1a;  /* Темный тоннель */
    color: #e6e6e6;             /* Светлый текст */
    font-family: 'Roboto', sans-serif;
}

/* Как красные знаки опасности */
.warning {
    color: #cc0000;
    border-left: 3px solid #cc0000;
    padding-left: 10px;
}

/* Как металлические двери */
.button {
    background: linear-gradient(to bottom, #4d4d4d, #333);
    border: 1px solid #666;
    border-radius: 3px;
    padding: 8px 15px;
    color: #fff;
}

Интерфейс выживания

Создадим интерфейс счетчика фильтров противогаза в стиле Metro

HTML:


<div class="gas-mask-ui">
    <div class="mask-display">
        <div class="filter-status">
            <span class="filter-text">ФИЛЬТРЫ:</span>
            <div class="filter-bar" id="filterBar"></div>
        </div>
        <div class="timer" id="filterTimer">05:23</div>
    </div>
    <button class="metro-btn" id="replaceFilter">Заменить</button>
</div>

CSS:


.gas-mask-ui {
    width: 300px;
    background: rgba(30, 30, 30, 0.8);
    border: 2px solid #555;
    padding: 15px;
    font-family: 'Courier New', monospace;
    color: #00ff00;
}

.filter-status {
    margin-bottom: 10px;
}

.filter-bar {
    height: 20px;
    background: linear-gradient(to right, #00aa00, #ff0000);
    margin-top: 5px;
    border: 1px solid #333;
    width: 100%;
    transition: width 0.5s ease, background 0.5s ease;
}

.timer {
    font-size: 24px;
    text-align: center;
    margin: 10px 0;
    color: #00ff00;
}

.metro-btn {
    background: #333;
    color: #fff;
    border: 1px solid #555;
    padding: 8px 15px;
    width: 100%;
    cursor: pointer;
    transition: background-color 0.3s;
}

.metro-btn:hover {
    background: #444;
}
